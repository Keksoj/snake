## Make it run

Be sure to [install Rust](https://www.rust-lang.org/tools/install) first. 
[Learn it](https://doc.rust-lang.org/book/) then. 
Best documentation I've ever read.

Clone the reposirory, go to its directory, run with cargo.

``` sh
git clone https://github.com/Keksoj/snake.git
cd snake
cargo run
```

Enjoy it, copy it, improve it.
